# 💰 Expense Tracker

A clean, minimal expense tracking app built with vanilla HTML, CSS and JavaScript.

## 🚀 Live Demo
https://expense-tracker-peach-three-29.vercel.app/

## ✨ Features
- Add expenses with name, amount and category
- Filter expenses by category (Food, Transport, Shopping, Bills, Health, Other)
- Real-time total calculation
- Delete individual expenses
- Data persists across page refreshes using localStorage
- Fully responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3 (CSS Grid, Flexbox)
- Vanilla JavaScript (localStorage, DOM manipulation, array methods)
- Deployed on Vercel

## 📁 Project Structure
```
expense-tracker/
├── index.html
├── style.css
└── script.js
```

## 💡 What I Learned Building This
- localStorage for persistent data storage
- JSON.stringify / JSON.parse for saving arrays
- Array methods — filter(), reduce(), map()
- Dynamic DOM manipulation with createElement
- CSS Grid for single-line form layout
